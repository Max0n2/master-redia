1. Гібридні архітектури мобільних застосунків на базі Flutter: порівняльний аналіз локальних та хмарних моделей машинного навчання
2. Редя Максим-Ігнат Володимирович
3. Чернихівський Євген, IT Степ Університет, доцент
4. Роботу присвячено дослідженню ефективності трьох архітектурних підходів до інтеграції моделей машинного навчання у мобільні застосунки, розроблені на платформі Flutter. У роботі розроблено тестовий Flutter-застосунок для задачі класифікації зображень, що реалізує три режими роботи: локальний (on-device inference з використанням моделі MobileNetV1), хмарний (REST API до моделі ViT-Base через сервіс HuggingFace) та гібридний. Гібридний режим реалізує адаптивну стратегію вибору джерела інференсу на основі confidence threshold: якщо впевненість локальної моделі перевищує встановлений поріг, результат приймається як кінцевий, інакше запит автоматично перенаправляється до хмарного API.
5. В папці проєкту є файл app-release.apk. Цей APK-файл потрібно будь-яким способом передати на телефон і запустити, або запустити через емулятор.
